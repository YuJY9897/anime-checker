# 애니 체크 개인정보처리방침

시행일: 2026.07.01.

애니 체크는 사용자의 애니 시청 기록을 기기 안에서 관리하기 위한 앱입니다.

## 1. 수집하는 정보

앱은 계정, 이름, 전화번호, 이메일, 위치, 연락처, 사진, 마이크 정보를 자동으로 수집하지 않습니다. 단, 사용자가 피드백 보내기 기능을 이용해 메일을 보내는 경우 이메일 주소와 작성한 내용이 개발자에게 전달될 수 있습니다.

앱 안에는 사용자가 직접 저장한 다음 정보가 기기 내부에 저장될 수 있습니다.

- 보관함 작품
- 보류 작품
- 찜한 작품
- 에피소드와 극장판 시청 기록
- 작품 메모
- 백업 생성 시각

## 2. 저장 방식

사용자 기록은 기본적으로 앱 내부 저장소의 로컬 JSON 파일에 저장됩니다.

사용자가 백업 기능을 실행하면 JSON 백업 파일을 직접 저장하거나 공유할 수 있습니다. 백업 파일을 외부 위치에 저장한 뒤의 관리는 사용자 책임입니다.

## 3. 네트워크 사용

앱은 다음 기능에서 인터넷을 사용할 수 있습니다.

- 애니 검색
- 작품 상세 정보 갱신
- 신작 애니 목록
- 애니 소식
- 포스터와 기사 이미지 표시

TMDB API 키는 앱 안에 저장하지 않고 Cloudflare Worker 프록시에서만 사용합니다.

피드백 메일은 사용자가 직접 전송 버튼을 눌렀을 때만 메일 앱으로 작성됩니다.

## 4. 제3자 제공

앱은 사용자의 보관함, 보류, 찜, 메모, 시청 기록을 개발자 서버에 동기화하거나 판매하지 않습니다.

검색, 신작 애니, 애니 소식 기능을 사용할 때 외부 데이터 제공자 또는 Cloudflare Worker로 요청이 전송될 수 있습니다.

## 5. 삭제

사용자는 앱 안의 전체 초기화 기능으로 보관함, 보류, 찜, 메모, 시청 기록을 삭제할 수 있습니다.

앱을 삭제하면 기기에 저장된 앱 내부 데이터도 함께 삭제됩니다. 사용자가 따로 저장한 JSON 백업 파일은 직접 삭제해야 합니다.

## 6. 데이터 출처

이 앱은 TMDB API를 사용하지만 TMDB가 보증하거나 인증한 앱이 아닙니다.

애니 소식은 외부 뉴스 출처의 제목, 요약, 출처, 날짜, 원문 링크를 표시할 수 있습니다. 기사와 이미지의 권리는 각 원 저작권자에게 있습니다.
