## 긴 설명

```text
애니 체크는 보고 있는 애니와 보고 싶은 애니를 정리하고, 새 화와 신작 정보를 확인할 수 있는 애니 시청 기록 앱입니다.

주요 기능
- 새로 볼 화가 있는 작품 확인
- 보관함, 보류, 찜 관리
- 시즌별 에피소드 시청 완료 기록
- 극장판과 영화 시청 기록
- 월별 신작 애니 탐색
- 요일 편성표 확인
- 애니 소식과 원문 보기
- JSON 백업 및 복원

사용자 데이터는 기본적으로 기기 내부에 저장되며, 사용자가 직접 JSON 백업 파일로 내보낼 수 있습니다. TMDB API 키는 앱에 포함하지 않고 서버 프록시를 통해 처리합니다.
```

## Data Safety 입력 가이드

앱 동작 기준 요약입니다. Play Console 질문 문구에 따라 최종 선택은 화면에서 다시 확인하세요.

- 계정 생성: 없음
- 로그인: 없음
- 광고: 없음
- 앱 내 구매: 없음
- 위치, 연락처, 사진, 마이크, 캘린더 권한: 사용 안 함
- Android 권한: `INTERNET`
- 보관함, 보류, 찜, 시청 기록, 메모: 기기 내부 로컬 저장
- JSON 백업: 사용자가 직접 내보내고 관리
- 검색, 신작 애니, 애니 소식, 포스터/기사 이미지: 네트워크 요청 사용
- TMDB API 키: 앱에 없음, Cloudflare Worker secret으로만 관리

권장 답변 방향:

- 사용자의 이름/전화번호/주소를 앱에서 자동 수집하지 않음
- 피드백 보내기를 사용하면 작성 내용과 선택 입력한 이메일 주소가 Cloudflare Worker/KV로 전송되어 저장될 수 있음
- 사용자의 위치를 수집하지 않음
- 사용자의 사진/동영상/오디오/연락처/캘린더를 수집하지 않음
- 앱 활동 중 보관함/찜/시청 기록은 로컬 저장이며 개발자 서버로 동기화하지 않음
- 네트워크 요청은 검색과 콘텐츠 표시 기능 제공 목적
- 피드백 전송은 사용자가 직접 보내기 버튼을 눌렀을 때만 실행되는 선택 기능
- 데이터 삭제 방법: 앱 안의 전체 초기화 또는 앱 삭제, 백업 파일은 사용자가 직접 삭제

주의:

- Play Console의 Data safety 답변과 개인정보처리방침 문구가 서로 달라지면 안 됩니다.
- 향후 광고, 로그인, 클라우드 동기화, 푸시 알림, AI 추천을 추가하면 Data safety와 개인정보처리방침을 다시 수정해야 합니다.

## 앱 콘텐츠 답변 메모

- 어린이 대상: 아니요
- 뉴스 앱 여부: 뉴스 전문 매체 앱이 아니라 애니 관련 소식 링크를 보여주는 보조 기능

- 건강, 금융, 정부, 도박, 데이팅, UGC 중심 앱: 아니요
- 콘텐츠 등급: 폭력적/성적/도박 콘텐츠를 직접 제공하지 않는 일반 정보/도구 앱 기준으로 답변
