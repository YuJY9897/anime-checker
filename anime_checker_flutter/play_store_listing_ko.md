# 애니 체크 Play Store 등록정보 초안

## 앱 이름

애니 체크

## 짧은 설명

신작 애니, 보관함, 찜, 시청 기록을 한곳에서 관리하세요.

## 긴 설명

애니 체크는 보고 있는 애니와 볼 예정인 애니를 정리하는 개인 시청 기록 앱입니다.

주요 기능:
- 새로 볼 화가 남은 작품 확인
- 보관함, 보류, 찜 목록 관리
- 작품별 시즌/에피소드 시청 기록 관리
- 신작 애니 월별 탐색
- 현재 방영중인 작품의 요일 편성표 확인
- 애니 관련 소식 확인
- JSON 백업/복원
- 다크 모드

애니 체크는 계정 가입 없이 사용할 수 있으며, 보관함과 시청 기록은 기본적으로 기기 내부에 저장됩니다. 검색, 신작 애니, 애니 소식은 외부 데이터 제공자와 Cloudflare Worker 프록시를 통해 불러옵니다.

이 앱은 TMDB API를 사용하지만 TMDB가 보증하거나 인증한 앱이 아닙니다.

## 태그 후보

애니, 애니메이션, 시청 기록, 신작 애니, 편성표, 찜 목록

## 스크린샷 추천 순서

1. 새 화
2. 보관함
3. 신작 애니 월별 보기
4. 작품 상세/에피소드
5. 애니 소식
6. 환경설정/백업
